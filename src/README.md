## Modules

- `eda.py`:  
  Functions for downloading, cleaning, and performing exploratory data analysis (EDA) on YFinance datasets, including calculation of daily returns, rolling statistics, and visualization of trends and volatility.

- `forecast.py`:  
  Implements time series forecasting models for Tesla (TSLA) stock prices, including ARIMA/SARIMA and LSTM models, with functions for model training, evaluation (MAE, RMSE, MAPE), and generating future forecasts with confidence intervals.

- `backtesting.py`:  
  Contains functions to simulate portfolio performance over a historical period, compute key performance metrics (total return, annualized return, volatility, Sharpe ratio, maximum drawdown), and compare multiple portfolios against benchmarks.
